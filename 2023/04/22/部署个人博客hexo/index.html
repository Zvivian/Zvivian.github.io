部署个人博客hexo

本地部署

第1步：安装node.js
完成安装后可以利用node -v npm -v 验证是否安装成功
第2步：npm安装cnpm
npm install -g cnpm –registry=https://registry.npm.taobao.org
-g表示全局安装，同样可以使用cnpm -v验证是否安装成功
第3步：安装hexo-cli
hexo -v 验证
第4步：建立一个文件夹并进入该路径
新建文件夹下保存博客信息，如果安装过程中出现不可修复问题，可直接删除该软件重新搭建
第5步：初始化hexo
hexo init
hexo s //启动博客，显示在localhost:4000端口打开，预览已经完成的博客
第6步：新建一篇博客
hexo n “博客名称”
输出创建成功提示。并且给出所在路径：Created: xxx\source_posts\博客名称.md
基于markdown语法格式编辑该.md文件
hexo clean // 清理一下数据库
hexo g // 生成博客

Github部署

准备工作：登录个人Github账户
第1步：新建仓库
注意：一定要以“Github账户名.github.io”命名仓库，之后可以直接在浏览器输入该名称访问博客
第2步：在hexo部署路径下安装hexo-deployer-git插件
cnpm install –save hexo-deployer-git
第3步：在同一路径下配置_config.yml文件
编辑状态下，找到最底部的Deployment：
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
type: git
repo: https://github.com/Zvivian/Zvivian.github.io 这里写入刚刚创建的仓库网址
branch: master 默认是master分支，也可以自己定义
第4步：将hexo博客部署到GitHub远端
hexo d
第5步：刷新GitHub仓库及博客页面
看到生成的博客网页
